# CodeAlpha Cyber Security Internship

This repository contains the tasks completed as part of the CodeAlpha Cyber Security Internship.

## Tasks

### Task 1 - Basic Network Sniffer
- Python-based network packet sniffer using Scapy.
- Captures and displays network packet information.

### Task 2 - Phishing Awareness Training
- PowerPoint presentation explaining phishing attacks.
- Covers phishing types, identification methods, and safety tips.

### Task 3 - Secure Coding Review
- Security review of the Basic Network Sniffer.
- Identifies vulnerabilities and provides recommendations.

## Tools & Technologies

- Python 3.13
- Scapy 2.7.0
- Visual Studio Code
- WPS Office
