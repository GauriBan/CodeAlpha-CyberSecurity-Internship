# Basic Network Sniffer

## Description

This project captures network packets using Python and the Scapy library.

## Features

* Captures live network packets
* Displays packet summaries
* Beginner-friendly project

## Requirements

* Python 3.13
* Scapy

## Installation

```bash
pip install scapy
```

## Run

```bash
python sniffer.py
```
